# Exercícios de MySQL - Atividades Práticas

Repositório contendo minhas resoluções dos exercícios de MySQL realizados como atividade prática.

## Sobre a Atividade

Este repositório contém minhas soluções pessoais para uma série de exercícios de MySQL, focando na aplicação prática dos conceitos aprendidos sobre bancos de dados relacionais.

## Estrutura do Repositório

```
desafioMysql/
├── desafio.sql    # Arquivo principal com TODAS as minhas soluções
└── README.md                # Este arquivo
```

## Conteúdo do Arquivo Principal

O arquivo `desafio.sql` contém:

- Criação do banco de dados "Clinica"
- Definição das tabelas (Ambulatorios, Medicos, Pacientes, Funcionarios, Consultas)
- Inserção de dados de exemplo
- Exercícios de atualização (UPDATE, DELETE)
- Consultas SELECT com diferentes níveis de complexidade
- Junções entre tabelas (INNER JOIN, LEFT JOIN)
- Subconsultas (ANY, ALL, EXISTS)
- Agrupamentos e ordenações (GROUP BY, ORDER BY)

## Tecnologias Utilizadas

- MySQL 8.0+
- MySQL Workbench

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/LorenaLira05/desafioMysql.git
```

2. Execute o arquivo SQL no MySQL:
```sql
SOURCE desafio.sql
```
